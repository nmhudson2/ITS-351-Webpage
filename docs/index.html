Using the script

First, create your account, then navigate to your profile under the settings page. Here you can find your API key, which will be used to construct your unique URL.
Do not share this key; any time it is loaded ona a page it will affect your metrics.



The URL for yout script is comprised of 3 segments: your API key, the list of metrics you want to capture, and finally an optional format parameter that is to be used
only when requesting all of the available data tied to the provided API key.



<script src="http://flowcone.com/users/api/<YOUR_API_KEY>.<FORMAT>.[METRICS_TO_CAPTURE]"></script>
